# Starter SaaS - Project Glossary

Cross-domain business terminology. Domain-specific terms are in each domain's GLOSSARY.md.

## Core Concepts

### User
An authenticated person in the system. Users can have roles in agencies.

### Agency
A business organization on Starter SaaS. Agencies have members with specific roles.

### Role
A permission set assigned to users within an agency. Values: `commercial`, `owner`.

### Membership
The relationship between a User and an Agency, including assigned roles.

## Technical Terms

### Convex
Backend-as-a-service platform providing real-time database and serverless functions.

### Better-Auth
Authentication library used for user login/signup.

### BDD (Behavior-Driven Development)
Testing approach using Gherkin scenarios to describe behavior in business language.

### Gherkin
Domain-specific language for writing BDD scenarios using Given/When/Then syntax.
